# Getting Started with BizTalk Server

BizTalk Server is a [publish and subscribe architecture](https://learn.microsoft.com/en-us/biztalk/core/publish-and-subscribe-architecture) that [uses adapters](https://learn.microsoft.com/en-us/biztalk/core/using-adapters) to receive and send messages, implements [business processes through orchestration](https://learn.microsoft.com/en-us/biztalk/core/defining-business-processes), and includes [management and tracking](https://learn.microsoft.com/en-us/biztalk/core/management-and-tracking-architecture) of these different parts. BizTalk Server also includes [trading partner management](https://learn.microsoft.com/en-us/biztalk/core/trading-partner-management-using-biztalk-server) for business-to-business messaging, [high availability](https://learn.microsoft.com/en-us/biztalk/core/planning-for-high-availability3) to maximize uptime, a development platform to [create your own components](https://learn.microsoft.com/en-us/biztalk/core/developing-custom-components), an administration console to [manage your artifacts](https://learn.microsoft.com/en-us/biztalk/core/operational-and-administrative-tasks-in-your-biztalk-environment), and business activity monitoring to manage [aggregations, alerts, and profiles](https://learn.microsoft.com/en-us/biztalk/core/using-business-activity-monitoring).
