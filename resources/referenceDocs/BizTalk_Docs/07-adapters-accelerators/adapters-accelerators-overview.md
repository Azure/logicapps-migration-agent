# Adapters and Accelerators in BizTalk Server

BizTalk Server includes different adapters and accelerators for you to create applications that receive data, and send data to different services and LOB systems.

This section describes the different adapters and accelerators available with BizTalk Server.

## Out-of-the-box Adapters

When you install BizTalk Server, you also install the built-in adapters that are ready to be used. Some of these adapters include File, FTP, MQ Series, Service Bus, Logic Apps, POP3, SharePoint, and more.

[Using Adapters](https://learn.microsoft.com/en-us/biztalk/core/using-adapters) lists all of them, and also shows you how to use each adapter.

## BizTalk Adapter Pack

The BizTalk Adapter Pack is included with BizTalk Server, and provides WCF-based adapters to connect your BizTalk Server to Oracle, SAP, Siebel, and SQL Server. You can also create your own WCF-based adapters using the WCF LOB Adapter SDK.

See [BizTalk Adapter Pack](https://learn.microsoft.com/en-us/biztalk/adapters-and-accelerators/biztalk-adapter-pack) to install and configure these adapters, step through tutorials and scenarios, create applications using the different adapters, and get a good idea of how WCF-based services process messages.

## Adapters for Enterprise Applications

These adapters are included with BizTalk Server. Use these adapters to connect your BizTalk Server with JD Edwards EnterpriseOne, JD Edwards OneWorld, PeopleSoft Enterprise, TIBCO Enterprise Message Service, and TIBCO Rendezvous.

See [BizTalk Adapter for Enterprise Applications](https://learn.microsoft.com/en-us/biztalk/adapters-and-accelerators/biztalk-adapters-for-enterprise-applications) to install and configure these adapters, step through tutorials and scenarios, create applications using the different adapters, and more.

## FileAct and InterAct

The BizTalk FileAct and InterAct Adapters are included with BizTalk Server, and provide connectivity between your BizTalk Server and the Society for Worldwide Interbank Financial Telecommunication (SWIFT) Secure IP Network (SIPN).

- The **FileAct adapter** provides secure and reliable exchange of files, and information pertaining to those files.
- The **InterAct adapter** provides secure and reliable exchange of individual structured financial messages.

See [FileAct and InterAct](https://learn.microsoft.com/en-us/biztalk/adapters-and-accelerators/fileact-interact/microsoft-biztalk-server-fileact-and-interact-adapters-documentation) to install and configure these adapters, step through some tutorials and scenarios, and get a good understanding of the architecture.

## HL7

The BizTalk Accelerator for HL7 is included with BizTalk Server, and provides connectivity between your BizTalk Server and health care computer applications based on the Health Level Seven (HL7) standard.

See [HL7](https://learn.microsoft.com/en-us/biztalk/adapters-and-accelerators/accelerator-hl7/microsoft-biztalk-accelerator-for-hl7-documentation) to install and configure the adapter, step through several tutorials and scenarios, learn how the adapter works, and use the different features, including schemas, acknowledgments, batching, validation, and more.

## RosettaNet

The BizTalk Accelerator for RosettaNet (BTARN) is included with BizTalk Server, and streamlines the development and deployment of RosettaNet standards-based integration solutions. BTARN supports the RosettaNet Implementation Framework (RNIF); which is an open network application framework that enables business partners to collaboratively run RosettaNet Partner Interface Processes (PIPs).

See [RosettaNet](https://learn.microsoft.com/en-us/biztalk/adapters-and-accelerators/accelerator-rosettanet/microsoft-biztalk-accelerator-for-rosettanet-documentation) to learn more about this accelerator, and using it with your BizTalk Server solutions.

## SWIFT

The BizTalk Accelerator for SWIFT is included with BizTalk Server, and provides connectivity between your BizTalk Server and the Society for Worldwide Interbank Financial Telecommunication (SWIFT) message formats.

Using the adapter with BizTalk Server, customers, partners, and system integrators can streamline the development, deployment, and support of integration solutions for core financial services application infrastructure and business processes.

See [SWIFT](https://learn.microsoft.com/en-us/biztalk/adapters-and-accelerators/accelerator-swift/microsoft-biztalk-accelerator-for-swift-documentation) to install and configure the adapter, step through some tutorials, and use the message repair, FIN response and FRR artifacts, and more.
