# Microsoft BizTalk Server FileAct and InterAct Adapters Documentation

The BizTalk FileAct and InterAct Adapters includes information that can help you install and configure the FileAct and InterAct adapters, learn about these adapters, and understand the architecture.

## Install and Configure

[Install and configure FileAct and InterAct adapter in BizTalk Server](https://learn.microsoft.com/en-us/biztalk/adapters-and-accelerators/fileact-interact/install-and-configure-fileact-and-interact-adapter-in-biztalk-server) lists the steps to install and configure the adapter, and also includes some sample messages and known install issues.

## Get Started

[Getting started with the FileAct and InterAct adapters](https://learn.microsoft.com/en-us/biztalk/adapters-and-accelerators/fileact-interact/getting-started-with-the-fileact-and-interact-adapters) introduces you to the SWIFTNet, FileAct, InterAct, and also includes a tutorial.

## Architecture

[Understanding FileAct and InterAct adapter architecture](https://learn.microsoft.com/en-us/biztalk/adapters-and-accelerators/fileact-interact/understanding-fileact-and-interact-adapter-architecture) describes the architecture of these different adapters.
