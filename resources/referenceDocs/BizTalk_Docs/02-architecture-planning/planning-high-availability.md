# Plan for High Availability

Many companies use BizTalk Server to process data that their business depends on. For these companies, the consequences of a prolonged downtime because of a hardware outage can mean diminished productivity and profitability. This section provides guidance for increasing availability of your BizTalk Server environment in order to maximize uptime of the BizTalk Server solution.
