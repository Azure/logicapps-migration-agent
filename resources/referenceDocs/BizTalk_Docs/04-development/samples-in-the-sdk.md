# Samples in the SDK

## Folder Paths

This section describes more than 30 samples included in the Microsoft BizTalk Server Software Development Kit (SDK). The section provides detailed information about each sample, including instructions for building the sample, how to run it, and what results to expect.

The samples documentation uses the path abbreviation `<Samples Path>` to denote the path to the samples in your installation. The default for the path can vary based on decisions made during installation. The following table provides the default path for common installation scenarios.

| Scenario | Default Path |
|---|---|
| Installation on 32-bit platform | `\Program Files (x86)\Microsoft BizTalk Server <VERSION>\SDK\Samples` |
| Installation on a 64-bit platform | `\Program Files (x86)\Microsoft BizTalk Server\SDK\Samples` |

> **Important:** For the sake of simplicity, the samples in this SDK, and this documentation, assume that you are using a single computer developer installation of BizTalk Server. If you do not perform a complete installation of BizTalk Server, some samples may not work properly.

> **Note:** All of the BizTalk Server SDK items are provided in English and are supported only for English-language installations of BizTalk Server.
