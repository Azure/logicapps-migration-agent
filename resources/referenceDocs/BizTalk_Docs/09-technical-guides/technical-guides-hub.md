# BizTalk Technical Guides

There are many products and services in the BizTalk family, including BizTalk Server, BizTalk Server on Microsoft Azure virtual machines, Microsoft Azure BizTalk Services (MABS), and Microsoft Azure App Service. This library includes guides and white papers that cover all areas, including performance, operations, ESB, and Microsoft IT (MSIT) scenarios.
